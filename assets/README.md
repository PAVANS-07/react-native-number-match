# Number Match Game — Assets

This directory should contain:
- `icon.png` — App icon (1024×1024)
- `splash.png` — Splash screen (1284×2778)
- `adaptive-icon.png` — Android adaptive icon foreground (1024×1024)
- `favicon.png` — Web favicon (48×48)

For development, Expo uses placeholder assets automatically.
For production APK, generate these with `expo prebuild` or provide your own images.
